# Quick Fabric

Quickly install a new version of fabric loader!

## Use

Clone it

```sh
git clone https://github.com/Iru21/quick_fabric
```

Install it

```sh
cargo install --path ./quick_fabric
```

And run! (Remember to have `~/.cargo/bin` in your `$PATH`)

```sh
quick_fabric [minecraft version]
```

## Example

```sh
quick_fabric 1.18.1
```
